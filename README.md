\## TicketPRO - Gestión de eventos online



La plataforma consta de un pequeńo frontend cómo introducción a los usuarios. 



A nivel lógico se utilizan las entidades: Proveedor, evento, categoría y ubicación. 



En este momento se encuentra desarrollado el panel de administrador de eventos con módulo para estadísticas y formulario para edición de perfil y creación de nuevos eventos.
